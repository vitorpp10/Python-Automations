PT-BR:

Para rodar esse projeto do zero, primeiro crie e ative um ambiente virtual Python. 
No terminal, dentro da pasta do projeto, execute python -m venv venv para criar o ambiente, depois ative com venv\Scripts\activate no Windows ou source venv/bin/activate no Linux/macOS. 
Em seguida, instale as dependências com pip install requests beautifulsoup4. 
Depois, suba o servidor HTTP simples apontando para a pasta site: uma opção é entrar na pasta site e rodar python -m http.server 3333, o que vai servir os arquivos estáticos em http://localhost:3333/. 
Com o servidor no ar e o ambiente virtual ativo, volte para a raiz do projeto e execute python main.py. 
O script vai fazer a requisição HTTP para o servidor local, baixar o HTML gerado por index.html, parsear o conteúdo com o BeautifulSoup e deixar os objetos parsed_html prontos para você inspecionar elementos, 
imprimir dados no terminal ou evoluir o experimento para um scraper mais completo.

EN-US:

To run this project from scratch, start by creating and activating a Python virtual environment. 
In the terminal, inside the project folder, run python -m venv venv to create the environment, then activate it with venv\Scripts\activate on Windows or source venv/bin/activate on Linux/macOS. 
Next, install the dependencies with pip install requests beautifulsoup4. 
After that, start a simple HTTP server pointing to the site folder: one easy option is to cd into the site directory and run python -m http.server 3333, which will serve the static files on http://localhost:3333/. 
With the server running and the virtual environment active, go back to the project root and execute python main.py. 
The script will send an HTTP request to the local server, download the HTML generated by index.html, parse the content with BeautifulSoup, and leave the parsed_html object ready for you to inspect elements, 
print data in the terminal, or evolve this exercise into a more advanced scraper.
