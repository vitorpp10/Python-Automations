This section of the repository contains three practice projects focused on **web scraping**, **HTTP requests**, and **HTML parsing**.  
They cover different difficulty levels and introduce essential concepts for interacting with web content using Python.

---

## Dragon Find Quotes

A command-line tool that scrapes quotes from `http://quotes.toscrape.com`.  
It demonstrates how to send HTTP requests, parse HTML with BeautifulSoup, handle pagination, and build an interactive terminal experience.

**Key concepts:** HTTP requests, HTML parsing, pagination, CLI workflow.

---

## Dragon URL Checker

A lightweight CLI program that checks whether a given website is online or offline.  
It uses `argparse` for argument handling and `requests` for connectivity testing.

**Key concepts:** argument parsing, error handling, request timeouts, status code evaluation.

---

## HTTP + BeautifulSoup Scraping Demo

A simple demonstration of scraping a locally hosted HTML page (`localhost:3333`).  
It shows how to read responses as text and bytes, parse HTML with different encodings, and extract structured data.

**Key concepts:** local server scraping, encoding handling, BeautifulSoup parsing basics.

---

## Purpose

These projects form the web scraping learning module for this repository.  
They explore the fundamentals of accessing, interpreting, and processing web-based content at multiple complexity levels, from basic parsing to interactive scraping utilities.

All exercises are designed to reinforce practical skills in HTTP communication, HTML structure understanding, and real-world automation techniques.
